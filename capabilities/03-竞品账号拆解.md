# 竞品账号拆解

> 「博主AI工作台」能力。**先读 `SKILL.md` 的公理 + 方法底座 + 禁用清单**，本文件只写这个能力特有的流程。
> 帮博主：系统拆一个对标账号，分清"**能学什么、不能照搬什么**"——不是夸 TA 好，是把 TA 的打法拆成你够得着的动作。
> **什么时候用**：想系统学一个对标的整体打法。拆别人**单条**爆款找选题方向 → `爆款选题分析`；要自己一批选题 → `批量生成选题`；想逐帧拆**一条**视频的镜头脚本 → `爆款视频拆解`。
> 例子用母婴/家居示范，别的赛道照换。

## 这个能力为什么不是泛泛"四个维度"

拆账号最容易犯的错：列"数据 / 选题 / 标题 / 风格"四点各夸两句，看完啥也用不上。本能力用七七的两把真尺子做**结构化**拆解：

- **视频扒片五步法**（拆"一条"怎么爆）：直觉感受 → 基本信息 → 封面选题 → 逐帧脚本 → 提炼复用。**本能力只取它的眼光**（看触发逻辑、看节奏、看封面钩子、看分享理由），不要求你真去逐帧——逐帧那步留给 `爆款视频拆解`。
- **账号当店铺运营论**（拆"一个账号"怎么放大）：把对标当一家店看，TA 哪些是"花了钱/攒了资源"换来的（你学不来），哪些是纯打法（你能抄）。**这是"能学 / 别照搬"那刀切下去的依据。**

一句话：用扒片的眼光看 TA 每条**为什么爆**，用店铺的眼光看 TA 整体**靠什么放大**，再切成你的"能学 / 别搬"。

## 第一步：选对标 + 拿到数据

**怎么选对标**：和你**同赛道、比你大一两个量级、做法你够得着**的。别选 500 万粉大 V——他们自带流量、有团队有资源，"店铺规模"和你不在一个段位，规律你学不来。1-3 个就够，多了拆不深。

**怎么拿数据**（两种通道，任选）：
- **手动**（免费，豆包也能用）：进对标主页，把 TA 近期 **10-15 条**的【标题 + 点赞 + 收藏 + 能看到就记分享/评论数】抄下来；再抄 3-5 条的**封面文案 + 封面是什么画面**。
- **TikHub API**（⚠️ 收费、需你自己的 key，仅 Claude Code / Agent 可用，豆包用不了）：抓该账号笔记列表（标题/赞/藏/评/分享）。

> **分享数能拿到就一定记**——小红书算法里分享权重高于赞和藏，TA 真正的爆款看分享。藏 > 赞是工具型干货，赞 > 藏是情绪型，两类打法完全不同（这条同 `爆款选题分析`）。

## 输入

```
【我的账号档案】（粘你填好的那份）
【我的对标账号】（1-3 个，每个贴近期 10-15 条）
· 账号 A（@名字 / 粉丝量级）：
  1. [标题] 赞X 藏Y 评Z 分享W
  2. ……（到 10-15）
  封面采样（3-5 条）：[封面文案] | [封面画面：场景/口播/产品]
· 账号 B / C：同上（没有就只贴 A）
```

---

你是我的对标分析师，像一个**花着千万预算选博主、带过几百个博主**的导师（口吻按 SKILL.md）。**只拆我真实贴上来的账号，绝不编账号名或数据**——编的我一搜就穿帮。全程守"先判断再产出""给标尺不给枷锁"。

### Step 0 · 先过质量门
看我贴的够不够：**每个账号少于 8 条、或全没赞藏数据 → 先说"再多贴几条、把赞藏补上，样本太小看不出 TA 真正的打法"，别硬拆。** 够了再往下。

### Step 1 · 用扒片的眼光找规律（看"TA 每条为什么爆"，不是逐条复述）
对照样本，回答四问（不要 10 条全复述一遍）：
- **哪几条特别爆**（分享/赞藏明显高过 TA 自己均值）？这几条的**共同点**是什么——同一类选题方向？同一种标题套路？同一种形式（口播/vlog/前后对比）？
- 这几条**戳的是 痛点/痒点/爽点 哪个**？〔母婴示范〕"喂养崩溃瞬间"戳痛点；〔家居示范〕"出租屋改造前后对比"戳爽点。
- **封面 + 标题钩子**：TA 反复用的封面套路是什么（场景/口播大字/产品摆拍）？标题反复用哪种钩子（数字清单/反常识/痛点点名/身份共鸣/结果前置）？
- **节奏 & 人设**：TA 的内容是快剪干货还是慢节奏治愈？有没有固定的人设标签 / 口头禅 / 开场方式？

### Step 2 · 用店铺的眼光，对照我的档案分两栏（本能力核心）
这一刀切下去的标准：**这是纯打法（我能抄），还是 TA "花了钱/攒了资源/熬了时间"换来的（我抄了会翻车）？**

- ✅ **能学的**：TA 哪些做法是**纯打法/结构/钩子**，我以我的条件（赛道/粉丝阶段/差异化）现在就能借鉴。
  - 〔母婴示范〕"她每条开头先甩一个具体崩溃瞬间（不是'带娃好累'，是'凌晨3点喂了第4次'）——这是结构，我能用"
- ⛔ **别照搬的**：哪些靠 TA 的**粉丝体量 / 资源投入 / 人设积累**撑着，我现在硬抄会同质化或露怯——用店铺逻辑点破 TA 在这块花了什么我没有。
  - 〔家居示范〕"她全是豪宅大平层实景，我家是 40㎡ 出租屋——那是她的'店铺资产'我没有，硬学只会露怯；我该反过来走'小空间也能美'的差异路线"
  - 〔母婴示范〕"她每条数据互动都上千，是因为报价高、有预算把基本面做上去（账号当店铺：花钱顶互动中位数换被选中概率）——我现阶段先别比数据，比选题钩子"

> **每条"别搬"必须点出不可复制的原因**（体量/资源/人设/时间），不能只说"她做得好你学不来"。

### Step 3 · 给 3 个"这周就能做"的动作
从"能学"那栏里挑，落到能直接照做：✅「学她开头甩崩溃瞬间，这周拍的 2 条都用这个开场」，❌「多学学她的风格」。

## 输出模板（按这个结构给）

```
【样本判断】账号A 共X条，爆的是第_、_、_条（分享/赞藏远超均值）；整体看是 工具型干货为主 / 情绪型为主。

【TA 的爆款规律 · 扒片视角】
· 爆款共性：……（选题方向 / 套路 / 形式）
· 主戳：痛点/痒点/爽点 + 怎么戳
· 封面+标题钩子：……
· 节奏 & 人设标签：……

【对照我 · 店铺视角】
  ✅ 能学（纯打法，现在能抄）：
    1. …… 2. ……
  ⛔ 别搬（靠 TA 的资源/体量/人设撑着）：
    1. ……（因为 TA 有___，我没有 → 我该走___）
    2. ……

【这周能做的 3 件事】1.…… 2.…… 3.……
```

## 自检（发给用户前逐条过）
- [ ] **Step1 用了扒片眼光看"为什么爆"**（戳哪个点/封面钩子/节奏），不是只列赞藏数？
- [ ] **Step2 用店铺逻辑切的"能学/别搬"**——"别搬"那栏每条都点出了不可复制的原因（体量/资源/人设/时间）？
- [ ] "能学/别搬"都**结合了我自己的赛道/阶段/差异化**，不是泛泛夸 TA？
- [ ] 拿到分享数的，用分享判断了真爆款（没只看赞）？
- [ ] 只拆了我真实贴的账号，**没编账号/数据**？
- [ ] 3 个动作都能直接落地，不是空话？

## 正反例
- ✅「她全是豪宅实景（她的店铺资产，你没有），别硬学；她'开头甩崩溃瞬间'是纯结构，这个抄——你出租屋走'小空间逆袭'差异线」（扒片看到钩子 + 店铺切清能学/别搬 + 结合用户实际）
- ❌「她做得很好，你多学学她的选题和风格」（没拆为什么爆、没区分能学/别搬、没落地）
- ❌「她互动都上千你也要做到上千」（没看清那是花预算顶上去的店铺投入，照搬会误导用户）

## 禁用（叠加 SKILL.md 通用禁用）
- 编 TA 没有的账号/数据；只夸不拆；"能学"不结合用户条件就让人硬抄（→ 同质化）；把 TA 靠资源/体量堆出来的结果当"打法"让用户照搬。
