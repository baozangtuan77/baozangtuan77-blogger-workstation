# 爆款视频拆解

> 「博主AI工作台」能力。**先读 `SKILL.md` 的公理 + 方法底座 + 禁用清单**，本文件只写本能力流程。
> 帮博主：拆**一条**爆款视频，提炼可复用的结构和打法（不是抄内容，是学机制）。
> 这一篇用的是七七拆爆款的真实框架——**视频扒片五步法**做结构化拆解骨架，逐帧/分镜深拆点名**视频分镜拆解方法论**，不是泛泛的"几维分析"。
> **什么时候用**：看到一条很爆的视频，想搞懂它怎么做的、能不能学。想从多条爆款找规律 → `爆款选题分析`；拆一个账号整体打法 → `竞品账号拆解`。
> ⚠️ **和 `03 竞品账号拆解` 分工别搞混**：03 拆**账号整体**（多条内容看打法/人设/节奏）；09 拆**单条视频逐帧**（一条视频抠到信号段、景别、钩子）。先用 03 选准对标，再用 09 抠它某条爆款怎么拍的。
> 例子用母婴/家居做〔示范〕，别的赛道照换。

## 第一步：拿到这条视频的逐字稿（打通这步，别只看封面瞎猜）

纯文字 AI 看不了视频画面，所以**先把口播转成文字**。四种拿法，挑一个：
1. **剪映（最简单免费）**：手机存下这条视频 → 导入剪映 → "文本 / 识别字幕" → 自动出全文 → 复制。剪映还能顺手用「智能镜头分割」帮你切信号段（下面第四步要用）。
2. **飞书妙记 / 通义听悟（自助版）**：上传视频，自动转写，导出文本。
3. **TikHub API**（⚠️ 收费、需自己 key，仅 Agent 模式）：抓这条笔记的视频转写。
4. **手动**：视频短就直接听打；或把"开头第一句 + 大致讲了啥 + 点赞/收藏/分享"告诉我。

> 把**逐字稿 + 点赞 + 收藏 + 分享**贴进来。**分享量尤其关键**——小红书算法里分享权重高于点赞收藏，真爆款的"分享理由"往往来自争议点或极强共鸣点，没有分享数据就少了判断"为什么爆"的最硬证据。有画面印象（封面、开头怎么拍、有没有特别的镜头）也一起说，拆得更准。

## 输入

```
【我的账号档案】（粘你填好的）
【这条爆款的逐字稿】：……
【点赞 / 收藏 / 分享】：……
【画面印象】（可选）：封面是啥、开头怎么拍的、有没有印象深的镜头
```

## 第二步 · 质量门

没有逐字稿、只有标题+点赞 → 让用户先按上面方法拿逐字稿，别硬拆。
只有点赞没有分享数据 → 提一句"补上分享量，分享才是真爆款的标志"，但有逐字稿就可以先拆。

## 第三步 · 用「视频扒片五步法」做结构化拆解（看机制，不复述内容）

> 这五步是七七扒片的标准动作。挨着走，每步一句话给结论，不照搬框架原文。

1. **直觉感受**：完整看（读）一遍，第一印象是什么节奏、什么情绪？在哪一段被抓住？——先建立整体感，后面都是验证它怎么被造出来的。
2. **基本信息**：内容类型（经验分享/好物种草/故事vlog/口播）、目标人群（精准圈定，如"0-3岁焦虑妈妈"）、时长、文案字数、用了哪些互动钩子/话题点。
3. **封面 + 选题**：封面是场景类/口播类/产品类？标题用了什么钩子（对比/警告/悬念/数字）？这条为什么能爆——靠话题点、稀缺感、反差还是情绪？
4. **逐帧拆脚本（核心步骤）**：按信号段（镜头切换点，剪映「智能镜头分割」能自动切）拆，每段记**文案 + 画面 + 景别 + 音效/花字**；再算两个硬指标：
   - **平均切换速度** = 时长 ÷ 信号数（参考：3.4秒≈快剪 / 5-6秒≈轻快 / 8秒+≈慢节奏）
   - **字数密度** = 文案字数 ÷ 时长
   > 想把这一步做到**导演级逐帧深拆**（景别节奏路径、运镜术语、光影色调、音画锚点、物理质感），用专门的 **视频分镜拆解方法论**——它把单条视频拆成九宫格 + 六维度的镜头语言。本能力点到信号段+节奏指标即可，深的全量留在加速器。
5. **总结提炼**：节奏归类（快/轻快/温柔/深情）、场景切换规律、情绪高潮在哪+怎么造的、可复用的结构/句式/钩子是什么。

> 〔示范·母婴〕"积食判断"那条：直觉=节奏偏快有紧迫感 → 人群=0-3岁焦虑妈妈 → 封面口播类+标题悬念钩"半夜哭别只当闹觉" → 逐帧:约18个信号段、均切3.8秒（快剪）、字数密度高 → 提炼:反常识开头+逐条信号清单+截图钩子。

## 第四步 · 提炼 3 个"我能复用"的点（抄机制不抄内容）

每个配我赛道一个落地示例。要抄的是"它为什么爆"那个机制（开头钩子结构 / 信号切换节奏 / 钩子设计），不是"它讲了啥选题"。

## 输出模板

```
【五步拆解】
1 直觉：节奏偏快·紧迫感，开头就被抓住
2 基本信息：口播·0-3岁焦虑妈妈·52秒·约140字·互动钩子=月龄征集
3 封面+选题：口播类封面 | 标题悬念钩 | 爆因=反常识反差
4 逐帧：约18信号段 | 均切3.8秒(快剪) | 字数密度2.7字/秒 | (深拆见分镜方法论)
5 提炼：反常识开头 + 逐条信号清单结构 + 截图/互动双钩子 | 分享>收藏·强共鸣型

【3 个可复用点】
1. [机制] → 我赛道示例：…
2. / 3. ……
```

## 自检（给用户前逐条过）
- [ ] 用户给了逐字稿了？（没有先让 TA 用第一步的方法拿，别硬拆）
- [ ] 走完了「扒片五步」、第四步算了切换速度/字数密度两个硬指标？
- [ ] 需要导演级逐帧时，点名了「视频分镜拆解方法论」而不是自己瞎拆镜头？
- [ ] 每步都说了"为什么有效"，不是复述内容？
- [ ] 3 个点是"可迁移的机制"，不是"抄它的选题"？
- [ ] 看了分享量、没只凭点赞判断爆款？没编用户没给的内容/数据？

## 正反例
- ✅「它开头先甩崩溃瞬间留人 + 均切3.8秒快剪不给走神空隙 → 这套'反常识钩子+快剪节奏'我也能用」——抄机制、有指标
- ❌「它讲辅食火了 → 我也讲辅食」——抄内容，换个选题就不灵
- ✅「逐帧发现它每个信号段都换景别、靠景别节奏维持新鲜感」——拆到镜头语言
- ❌「它镜头很酷」——说不清为什么，等于没拆

## 禁用（叠加 SKILL.md 通用禁用）
- 没逐字稿就凭标题硬拆；编用户没给的内容/数据；只复述不提炼机制。
- 别只看点赞判断爆款（分享才是真爆款标志）；别把九宫格/六维度全量方法论整篇搬进来——本能力只点框架名+一句，深拆留加速器。
