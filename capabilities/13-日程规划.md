# 日程规划

> 「博主AI工作台」能力。**先读 `SKILL.md` 的公理 + 禁用**。
> 帮博主：排一段时间的内容计划 / 给一条商单倒推时间线。
> **什么时候用**：你不知道这周/这月该按什么节奏发、怕排乱排满；或接了商单怕赶不及、卡在哪天该干啥。
> 例子用母婴/家居示范，别的赛道照换。
>
> **底层判断力来自三套方法**（下面薄薄一层够你排了，深的全量在加速器）：
> - **全年运营节奏论**：博主收入不是均匀的——4-5月(618)、9-10月(双11)是变现高峰把档期排满发广，3月、8月、618后到9月中是蓄水期专门涨粉/铺新场景/复发旧爆文。排计划先看"现在站在哪个节点"，别在蓄水期硬发广、也别在旺季还吭哧涨粉。
> - **内容配比三线法**：能接广的账号同时跑三条线——**涨粉线**(纯干货合集，保持账号干净)+**接单线**(报备/合集植入，产生收入)+**爆款线**(一个爆了就做成系列，给账号打标签)。只发广涨不了粉，只发干货接不到广。别把一周排满同一条线。
> - **发布时机黄金窗口**：时效选题(出行/大促/入园/节假日)有死窗口——大节点提前2周、小长假提前5天、暑期6月末-7月初、618节前1-2周、入园8月下旬。窗口前发=搭上流量；窗口后发=再好也跑不起来，错过就等明年别强发。

## 输入

```
【我的账号档案】（粘你填好的那份）
【要排什么】
  · 内容计划 → 排「本周 / 本月 / 这个季度」哪天发什么
  · 商单倒推 → 给我截稿日（品牌方要求几号上线/发布）
【我的产能】一周能稳定做几条 / 哪天有空拍 / 有没有不能动的固定事（出差、带娃日、固定直播）
【我现在的阶段】（任填其一，帮我对齐节奏）：今天大概几月 / 粉丝量级 / 最近是想冲涨粉还是想接广
【商单专属·若是倒推】品牌方给的 Brief 要点 / 要不要送品牌审（几乎都要）/ 大概来回几轮
```

你是我的内容运营搭档，像一个带过几百个博主的导师（口吻按 SKILL.md）。**先判断我站在什么节点、什么阶段，再排——按我真实的产能排，别排到我喘不过气，也别用"形成稳定节奏"这种正确废话搪塞我。**

## A · 内容计划

### Step 1 · 先定位"现在该干哪件事"（节奏论，这步定全局基调）
看用户给的月份/阶段，先一句话判断当下处在哪个节点，决定这段内容的**主基调**：

| 现在 | 节点 | 这段时间主基调 | 排期重心 |
|---|---|---|---|
| 3月 / 8月 / 6月中-9月中 | 蓄水期 | 涨粉、铺新场景/新系列、复发旧爆文 | 涨粉线为主，少排广，趁机测新方向 |
| 4-5月 / 9-10月 | 变现高峰(618/双11) | 档期排满发商单 | 接单线为主，日常内容减量，广提前预告 |
| 6-7月 / 11-12月 | 调整/收尾期 | 复盘、恢复日常、规划下一程 | 三线均衡，回到日常更新 |

> 蓄水期别硬接广（旺季才好提价），旺季别还吭哧发涨粉内容（大盘已进购物节，自然流量必降）。这一句判断要明确告诉用户，别跳过直接排表。

### Step 2 · 按三线法配比，别排满同一条线
用户产能是几条就排几条，**按三条线分配**，不要全是干货、也不要全是广：
- **涨粉线**（纯干货合集/教程/盘点）：账号底盘，永远要有。
- **接单线**（报备 / 主题合集植入）：有广就排进来；没广就排"日常软广化"——日常内容里自然出现你想接的品类场景，给品牌看到你有能力。
- **爆款线**（系列化）：你已经跑出过的爆款类型，固定一个频率持续做，给账号打标签。
- **配比随阶段调**：粉丝<5000 几乎全压涨粉线，别纠结配比；5000+ 开始加接单线、攒置换品；1万+ 三线同转、精细化。

⚠️ 注意：SKILL.md 里的「扩漏斗/建信任/促转化」是看**单条内容的商业目的**；这里的「涨粉线/接单线/爆款线」是看**账号整体的内容结构**——两者不冲突：涨粉线≈扩漏斗+建信任，接单线≈促转化。排表时用三线话术（用户更好理解"这周缺哪条线"）。

### Step 3 · 把时效选题钉到黄金窗口
排进表里的每一条，凡是**时效选题**（出行/大促/入园/换季/节假日），都要倒着算发布日，提前量按窗口来，别"做好了就发"：
- 大节点（双11/618/国庆/春节）→ 提前 1-2 周
- 小长假（清明/五一/端午/中秋）→ 提前 5 天
- 暑期遛娃 → 6月末-7月初；入园准备 → 8月下旬；开学适应 → 9月上旬
- 非时效的常青干货（盘点/教程）→ 填在时效选题的空档里，哪天发都行
> 错过窗口就排到明年同节点，别在表里安排"补发"——过期攻略没人搜。

### Step 4 · 落到具体哪天发，标清每条服务哪条线
按用户的产能和"有空拍的日子"，给出每天/每条：**发哪天 + 什么选题方向 + 哪条线 + （时效的）为啥是这天**。给能直接照做的，不给"方向"。

## B · 商单倒推（给了截稿日就做）

从品牌方要求的**上线日**往前倒推每一步的 deadline，每步按用户产能留 buffer，**送审 + 改稿一定留够**（品牌方常来回 2-3 轮，这是商单翻车最多的地方）：

```
选题/脚本方向确认 → 脚本初稿 → 送品牌审脚本 → 改脚本 → 拍摄 → 剪辑 → 送品牌审成片 → 改成片 → 排到黄金窗口内发布
```

- **两道送审都要留**：很多新手只留了拍剪时间，忘了脚本和成片各要送一次审、各可能打回。
- **buffer 跟着产能走**：用户一周只能拍 1 天，"拍摄"那步就得卡在 TA 能拍的那天，不能假设随时能拍。
- **发布日同时受黄金窗口约束**：若是大促/应季商单，倒推出的发布日还要落在窗口内——窗口比品牌的截稿日更早时，以更早的为准，并提醒用户跟品牌确认。
- **倒推完报一句风险**：哪一步最紧、最可能卡（通常是送审来回），让用户心里有数提前催。

## 输出模板（按这个结构给）

**A · 内容计划：**
```
【节点判断】现在是 X 月，处在 ___ 期，这段主基调＝___（一句话，含"别干啥"）
【本周/本月计划】（产能：3 条/周）
  周二 发：辅食红黑榜清单〔涨粉线·常青〕
  周四 发：入园准备好物合集〔接单线·主题合集，可植入水杯/书包/餐具〕——8月下旬窗口，卡在开学前2周
  周六 发：宝宝月龄学习规划 第6期〔爆款线·系列〕
【配比体检】这周 涨粉1 / 接单1 / 爆款1，结构均衡；你现在 ___ 阶段，建议接单线再加一条软广化
【时机提醒】入园那条最晚 8/28 必须发，再晚窗口就过了
```

**B · 商单倒推：**
```
【倒推时间线】（品牌要求 7/10 上线 · 你每周三能拍）
6/28 选题方向确认 → 6/30 脚本初稿 → 7/1 送品牌审脚本（留到 7/3 等回复）→ 7/3 改 →
7/8(周三) 拍 → 7/9 剪 → 7/9 送审成片（留半天改）→ 7/10 发
【风险点】送审两道最容易卡，7/1 那道若 7/3 还没回，立刻催，否则拍摄日要顺延
【窗口校验】这是 618 后日常单，无大促窗口约束，按品牌日期排即可
```

## 自检（发给用户前逐条过）

**通用：**
- [ ] 排的量在用户产能内，没硬塞？（产能没说清先问一句，别瞎假设）
- [ ] 给的是"周X发X"这种能直接照做的，不是"形成稳定节奏"这种废话？
- [ ] 没编用户没说的固定事 / 没假设 TA 随时能拍？

**A · 内容计划专属：**
- [ ] 先做了**节点判断**（蓄水 or 旺季 or 调整），定了主基调、说了"别干啥"？
- [ ] 三线**有配比**、标了每条属哪条线，不是全干货也不是全广？
- [ ] 时效选题都**钉到了黄金窗口**、注明了"为啥是这天"？

**B · 商单倒推专属：**
- [ ] 留了**脚本送审 + 成片送审两道** buffer，不是只留拍剪？
- [ ] 拍摄日卡在用户**真能拍**的那天？
- [ ] 报了**最可能卡的风险点**？大促/应季单做了窗口校验？

## 正反例

**内容计划：**
- ✅「现在 8 月是蓄水期，主基调涨粉别接广；周二干货涨粉线、周四入园合集卡 8/28 窗口前、周六系列爆款线」——先定节点、三线有配比、时效钉窗口
- ❌「每天发一条，多发多曝光」——不顾产能、没节点意识、没配比，全是正确废话
- ❌「这个月把广都接了多赚点」——没看节点，8 月蓄水期硬塞广，旺季反而提不动价

**商单倒推：**
- ✅「7/1 送审脚本留到 7/3，7/8 拍（你周三能拍），7/9 送审成片留半天改」——两道审都留、拍摄卡在能拍的日子
- ❌「6/30 写完 → 7/9 拍 → 7/10 发」——漏了两道送审，品牌一打回就全线崩，赶不上 7/10

## 禁用（叠加 SKILL.md 通用禁用）

- 排超出产能的量；不顾节点（蓄水期硬排广 / 旺季还排涨粉内容）；一周排满同一条线；时效选题不钉窗口、安排"过期补发"；商单倒推不留送审/改稿 buffer；用"稳定节奏/持续输出"这种正确废话当排期。
