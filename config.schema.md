# 《我的账号档案》字段说明

这份档案是整个工作台的 config——它让 AI"认得你"，给的建议才贴你，不然全是放之四海的废话。**填一次，每次用能力都粘在最前面。**

| 字段 | 填什么 | 为什么重要 |
|---|---|---|
| **赛道** | 母婴 / 家居 / 你的赛道（**二选一别混**）| AI 守赛道边界、举例落到你的赛道 |
| **我是谁** | 身份背景（俩娃妈妈、设计师…）| 决定人设和可信度 |
| **主做方向** | 你内容的主线（辅食 / 收纳…）| 选题、脚本都围着它转 |
| **受众** | 给谁看（越具体越好：0-3 岁新手妈妈）| 决定戳谁的痛点 |
| **差异化** | 你和别人不一样在哪 | 立人设、防同质化 |
| **粉丝阶段** | 刚起号 / 卡住 / 想变现 | 决定你现在该涨粉还是变现 |
| **文风样本** | 一条你真实发过的文案 | AI 学你的说话方式，产出像你 |
| **变现目标** | 接商单 / 卖货 / 引流 / 涨粉 | 每条内容都会问"怎么帮你这个目标" |

> 填不准没关系——先填一版，用「商业定位诊断」「内容数据复盘」跑几次，再回来更新档案，越用越准。
