# 直播转短视频

> 「博主AI工作台」能力。**先读 `SKILL.md` 的公理 + 方法底座 + 禁用**，本文件只写本能力流程。
> 帮博主：把一场直播，**端到端**变成可发的短视频——选片段 → 写句级剪辑脚本 → 指导你/剪辑师在剪映里剪出来。一场直播能拆好几条。
> 写句级脚本时点名两个七七的真实框架：脚本骨架用「爆款脚本结构十三式」、节奏指引用「剪辑节奏三层论」——不是套路化模板。
> 例子用母婴/家居示范，别的赛道照换。

## 第一步：拿到带时间的逐字稿（打通这步，别卡在这）

剪辑要按时间点定位，所以先要**带时间戳的逐字稿**。三种拿法：
1. **剪映（最简单免费）**：把直播录屏导入剪映 → "文本/智能字幕/识别字幕" → 自动出全程字幕（带时间）→ 导出字幕文本。
2. **飞书妙记 / 通义听悟（自助版）**：上传录屏，自动转写带时间轴，导出。
3. **手动**：没录屏只有大致内容，就把"几分几秒讲了啥"大致列出来。
> 把逐字稿贴进来（带时间最好）。

## 第二步：内容价值地图（先选，等你确认，别急着写脚本）

通读逐字稿，找出**能独立成篇**的话题模块。识别标准（必须"完整观点" + 至少 1 条加分）：

| 维度 | 说明 |
|---|---|
| 完整观点（必要）| 论点+论据+结论，单拎出来能成立 |
| 反直觉/争议 | 推翻常见认知，自带传播 |
| 具体数据/案例 | 真实数字、学员/自己的故事 |
| 可执行方法 | 有步骤，能直接抄 |
| 情绪共鸣 | 戳中受众真实痛苦场景 |

**排除**：个案咨询、闲聊暖场、已发过的、脱离你受众的。

每个话题给：⭐ 潜力（情绪共鸣/实操价值/争议性/受众匹配 四维）+ 时间段 + 钩子金句 + 一句摘要。

**⚠️ 停，等我确认要做哪几条，再写脚本。**

## 第三步：写「句级剪辑脚本」（剪辑师/你照着剪的施工图）

对选定的每条，先定骨架，再按**爆款逻辑链**写到句级，最后给每段标上**节奏指引**。

### ① 先定骨架（用「爆款脚本结构十三式」选最对路的一式）

> 直播切片大多是经验/观点分享，从「十三式」里挑最对路的一式当骨架，一句话说为什么——和 `短视频脚本编导` 一致，别用我编的类型。常用：**结论先行-逐条论证（式10）**（观点纠偏，先抛反常识结论）/ **痛点清单干货（式4）**（一段直播里讲了好几个点，逐条拆）/ **角色代入-逐条揭秘（式11）**（行业揭秘/避坑）。拿不准就默认逐条论证型——最稳。

### ② 按爆款逻辑链写到句级（7步骨架，每段精确到句）
1. **钩子**（开头 5 秒内建完整悬念：金句/反差/痛点直给）
2. **真相揭秘**（论据撑结论，不是再说一遍结论）
3. **痛点强化**（数字大字报 / 对比）
4. **破局方案 1**（方法+案例）
5. **破局方案 2**（另一种，可省）
6. **情绪高潮**（金句+花字全屏）
7. **金句回扣**（呼应开头，闭环）

**❌ 砍掉的开头废话**：过渡词（"那么对于…"）、反问（"体现在哪呢？"）、自问自答、上下文连接（"刚刚我们讲了"）。

每段标：原片时间段 + 保留的原话 + 删哪句废话 + 🔤 花字/数字大字报指引 + 🎵 音效点 + 字幕高亮关键词。**保留你直播的真实金句/数据原话，不改成书面腔。**

### ③ 给每段标节奏（用「剪辑节奏三层论」让脚本的节奏指引落到三层，不只标"这里快一点"）

> 节奏不是靠剪得多碎，是三层叠加——**句级脚本要在每段同时给到这三层的指引**，剪辑师才照得出节奏感：
> - **故事节奏层（切换密度）**：这段是快还是慢？直播切片多半同一场景（你在镜头前讲），靠**景别切换**（近景↔特写）替代场景切换来提速；钩子段、情绪高潮段切换密，铺垫段可慢——一段一个判断，别整条匀速。
> - **文本节奏层（句式）**：留原话时优先留**短句**（≤7字最好），短句处才好做切换点；遇到直播里的大长句，标注"做逐渐放大缩小、别硬切碎"——这是脚本层就能定的，别留给剪辑现想。
> - **剪辑技巧层（16 手法点名）**：每段花字/音效/特效**点名具体哪一类**，别只写"加花字"。常用：结构段用**大花字/背景+花字**做章节划分；**音效**最易漏、每个花字出现/重点处/转场都要配（搜"综艺"音效）；短句关键词用**瞬间放大**、长句用**逐渐放大（关键帧）**；情绪段配**贴切的人物特效**（找不到贴切的宁可不加）。

**节奏自检三句**：① 这段切换密度配它的情绪了吗（钩子/高潮要密）？② 留的原话是短句、有切换点吗？③ 花字/音效/特效点名到具体哪一类、且贴切吗？

末尾给：标题 3 选（不同公式）+ 封面方案。

**⏱ 时长**：单条 2-3 分钟最优；3-5 分钟警告完播；>5 分钟必须拆条。**等我确认脚本再往下。**

## 第四步：怎么剪出来（剪映工作流，打通到成品）

告诉我照脚本在剪映里怎么做：
1. 按脚本时间段，把选中片段剪出来、按"剪辑顺序"拼接（不一定等于原片顺序）。
2. 加字幕（白字 + 黄色关键词高亮 + 底部）。
3. 加结构标签（左上角 场/转/术 或 1/2/3）。
4. 按脚本第三步②的指引加数字大字报花字、③的节奏指引落每段的景别切换/放大/特效。
5. 段落切换加转场音效；金句处加"叮"——音效最易漏，对照脚本逐段配齐。
6. 需要就加 BGM，导出。
> 没时间自己剪？这份句级脚本可以直接发给剪辑师，是完整施工图（结构+逐句+三层节奏都标好了）。

## 自检（给用户前逐条过）
- [ ] 价值地图先给了、等用户确认了，没自顾自写脚本？
- [ ] 骨架从「十三式」点名选了某一式（说了为什么），不是泛泛的"干货型"？
- [ ] 每条能独立看懂，不依赖直播上下文？
- [ ] 开头 5 秒砍光废话、能独立抓人？结尾金句回扣开头？
- [ ] 每段都给了「剪辑节奏三层论」三层指引（切换密度 / 短句切换点 / 点名具体花字音效特效），不是只写"这里快一点"？
- [ ] 音效逐段配齐了（最易漏的一层）？特效都"贴切"、没硬加？
- [ ] 真实金句/数据原样保留，没改成书面腔？
- [ ] 给了剪映落地步骤，不是脚本写完就完（打通到成品）？

## 正反例
- ✅ 段1保留原话"我当时就懵了"+前面加钩子"接了个商单差点翻车"，删掉"那么我前几天说"
- ✅ 节奏标"段3情绪高潮：切换密+人物特效(委屈脸)+全屏大花字+'叮'音效"——三层都落到具体手法
- ❌ 把"我当时就懵了"改成"我感到十分意外"——活人感改没了
- ❌ 节奏只写"这段快一点、加点花字"——没分层、没点名，剪辑剪不出来

## 禁用（叠加 SKILL.md）
- 编直播里没说的数据/案例；把口语金句改成书面腔；价值地图不确认就写脚本；单条 >5 分钟还不拆。
- 节奏指引只写"快/慢"不分三层、花字音效不点名到具体哪一类；硬加不贴切的特效。
- 框架只嵌「名+怎么用」就够，别整篇搬全量方法论（全量在付费加速器）。
